## Installation

1. Clonez le dépôt :
   git clone https://github.com/sandrinah12/figmaland-Nuxt3-Tailwind.git


2. Accédez au dossier du projet :  
   cd figmaland-Nuxt3-Tailwind

2. Installez les dépendances :  
   npm install

## Lancement du projet

En mode développement :  
- npm run dev
